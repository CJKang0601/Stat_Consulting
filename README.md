# Statistical Consulting

This repository collects my NCKU statistical consulting coursework and final
project materials.

## Repository layout

- `hw1/`
  - `source/`: homework source files
  - `report/`: rendered report output
  - `data/`: small supporting dataset used in the homework
- `hw2/`
  - `source/`: homework source files
  - `report/`: rendered report output
- `project/`
  - `scripts/`: R and Python project scripts
  - `reports/`: project slides and report-style materials
  - `papers/`: reference reading materials
  - `figures/`: generated figures and visualizations
  - `submissions/`: CSV submissions
  - `data/`: lightweight project data files
  - `archives/`: compressed dataset archive

## Data note

The original Kaggle `train.csv` for the store sales project is larger than the
single-file GitHub upload limit, so it is intentionally not tracked here.
The compressed archive and the remaining project files are kept instead.

## Purpose

This cleanup focuses on making the repo easier to browse and preserving the
course outputs without changing the statistical work itself.
